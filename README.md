# 💍 Wedding Website

A beautiful wedding website built using HTML, CSS, and JavaScript, customized from a premium wedding theme. This website provides guests with all the important details about our special day, including event information, venue details, photo galleries, and RSVP information.

## ✨ Features

- Elegant and responsive design
- Date countdown
- Wedding event information
- Photo gallery
- Venue and schedule details
- RSVP section
- Smooth animations and interactive elements
- Mobile-friendly layout

## 🛠 Built With

- HTML5
- CSS3
- JavaScript
- Bootstrap
- Customized Wedding Theme

## ❤️ About

This website was created to share our wedding journey and provide guests with all the information they need for our celebration.
